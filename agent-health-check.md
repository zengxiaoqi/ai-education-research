# Agent 健康巡检脚本

## 任务说明
每小时执行一次，检查所有 Agent 任务处理状态。

## 执行步骤
1. 调用 `sessions_list` 获取所有活跃会话
2. 调用 `subagents action=list` 检查子代理状态
3. 识别异常：
   - 会话运行时间过长（>30 分钟）
   - 错误状态
   - 卡住无响应
4. 对异常会话进行干预（steer 或 kill）
5. 生成报告
6. 使用 `message` 工具发送报告到：
   - telegram 通道
   - qqbot 通道

## 报告格式
```
🔍 Agent 健康巡检报告
时间：{当前时间}
总计检查：{数量}
异常数量：{数量}
已处理：{数量}

详情：
{列表}
```
