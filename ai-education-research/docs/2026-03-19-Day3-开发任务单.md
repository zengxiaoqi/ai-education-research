## 4) 接口契约（最小版）

## 4.1 提交作答
`POST /api/submissions`

```json
{
  "assignmentId": "asgmt_123",
  "studentId": "stu_001",
  "answer": "42",
  "reasoning": "我先根据..."
}
```

响应：
```json
{
  "submissionId": "sub_999",
  "status": "submitted"
}
```

## 4.2 诊断结果
`GET /api/submissions/:id/diagnosis`

```json
{
  "misconceptionType": "formula_misuse",
  "evidence": "将平方差误写为...",
  "suggestion": "先复习公式 A，再做题 B",
  "confidence": 0.61,
  "needsReview": true
}
```
