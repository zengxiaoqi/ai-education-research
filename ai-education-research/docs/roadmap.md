# 产品路线图 (Roadmap)

基于 `product-upgrade.md` 和 `feature-backlog.md`，明确各阶段里程碑与交付时间。

---

## 概览

| 阶段 | 周期 | 核心主题 | 关键里程碑 |
|------|------|----------|------------|
| Phase 1 MVP+ | 2026 Q2-Q3 | 批改评阅闭环 | Beta 发布 |
| Phase 2 个性化 | 2026 Q4 | 个性化学习 | 公开版发布 |
| Phase 3 平台 | 2027 H1 | AI 原生平台 | 企业版准备 |
| Phase 4 生态 | 2027 H2+ | 开放生态 | 生态初具 |

---

## Phase 1: MVP+ (2026 Q2-Q3)

**目标**：打造可用、易用的核心批改评阅闭环

### 时间线

```
Q2 (4-6月)          Q3 (7-9月)
├── Month 1-2      ├── Month 7-8
│   ├─ 技术方案确定  │   ├─ 教师审核台
│   ├─ 开发环境搭建  │   ├─ 学生端
│   └─ 基础 CRUD    │   └─ 基础数据面板
├── Month 3-4      ├── Month 9
│   ├─ AI 批改引擎  │   ├─ 内测
│   └─ 班级/作业    │   └─ Beta 发布
└── Month 5-6
    └─ 集成测试
```

### 里程碑

| 里程碑 | 时间 | 交付 | 验收标准 |
|--------|------|------|----------|
| M1.1 技术就绪 | 4月底 | 技术方案 + 开发环境 | 技术选型确认 |
| M1.2 基础能力 | 5月底 | 用户/班级/作业 CRUD | 核心流程跑通 |
| M1.3 AI 批改 | 6月中 | AI 评分 + 基础评语 | 准确率 ≥ 80% |
| M1.4 端到端 | 7月底 | 教师审核 + 学生提交 | 完整流程演示 |
| M1.5 Beta | 9月 | 5-10 所学校内测 | 稳定可用 |

### 资源估算

- 前端：2 人
- 后端：2 人
- AI/算法：1 人
- 产品/设计：1 人

## Phase 2: Personalized Learning (2026 Q4)

**目标**：从"批改工具"升级为"学习平台"

### 时间线

```
Q4 (10-12月)
├── Month 10-11
│   ├─ 能力图谱 + 错别字诊断
│   ├─ 自适应推荐 v1
│   └─ 写作素材库
├── Month 11-12
│   ├─ 家长端
│   ├─ AI 写作助手
│   └─ 公开版 Release
└── 运营启动
```

### 里程碑

| 里程碑 | 时间 | 交付 | 验收标准 |
|--------|------|------|----------|
| M2.1 能力诊断 | 10月中 | 能力图谱 + 错别字 | 4维能力可视化 |
| M2.2 自适应 | 11月初 | 练习推荐 | 推荐准确率 ≥ 70% |
| M2.3 素材库 | 11月中 | 素材库 | 覆盖主流题材 |
| M2.4 公开版 | 12月底 | 正式发布 | DAU ≥ 1000 |

## Phase 3: Adaptive Platform (2027 H1)

**目标**：AI 原生的自适应学习平台

### 时间线

```
2027 H1
├── Q1 (1-3月)
│   ├─ LLM 大模型辅导
│   ├─ 知识图谱 v1
│   └─ AI 数字人 Demo
├── Q2 (4-6月)
│   ├─ 跨学科作文
│   ├─ AI 组卷
│   ├─ 区域学情
│   └─ 企业版准备
```

### 里程碑

| 里程碑 | 时间 | 交付 | 验收标准 |
|--------|------|------|----------|
| M3.1 LLM 辅导 | 2月底 | DeepSeek 对话辅导 | 响应质量评分 ≥ 4/5 |
| M3.2 知识图谱 | 3月中 | 知识点关联 | 覆盖 P1-P3 知识点 |
| M3.3 数字人 | 3月底 | 虚拟导师 Demo | 可演示 |
| M3.4 企业版 | 6月底 | 私有化部署 | 签约 2+ 客户 |

## 风险预案

| 场景 | 预案 |
|------|------|
| AI 准确率不达标 | 多模型 ensemble + 人工兜底 |

| 人才不足 | 外包 + 实习生补充 |
| 监管政策变化 | 法务合规前置 |
