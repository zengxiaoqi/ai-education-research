assistant: [[reply_to_current]] 这个报错是因为你在 **bash** 里执行了 **PowerShell 语法**：

- `iex` / `irm` 是 PowerShell 命令  
- 你当前提示符是 `zxq@...:~$`，这是 bash

你有两种解决方式：

1. **切到 PowerShell 再执行**
```bash
pwsh
```
然后在 PowerShell 里跑：
```powershell
iex (irm 'https://raw.githubusercontent.com/QuantumNous/new-api-docs/refs/heads/main/helper/codex-cli-setup.ps1')
```

2. **直接在 bash 里调用 pwsh 一次性执行**

```bash
pwsh -NoProfile -Command "iex (irm 'https://raw.githubusercontent.com/QuantumNous/new-api-docs/refs/heads/main/helper/codex-cli-setup.ps1')"
```

另外你贴的命令里 `codex` 和 `-cli-setup.ps1` 之间有换行，记得 URL 必须是连续的一整行。
